# 📊 Sistem Pelacak Langganan Software Perusahaan

Aplikasi web responsif yang dirancang untuk membantu tim operasional dan manajemen melacak siklus perpanjangan aplikasi, perangkat lunak, dan langganan alat bantu perusahaan agar lebih terorganisir dan efisien.

🔗 **[Lihat Live Demo Aplikasi Di Sini](https://pt-lead-geeks-indonesia.vercel.app)**

## 💡 Nilai Bisnis & Solusi
Aplikasi ini dibuat untuk menyelesaikan masalah umum di perusahaan: kelupaan memperpanjang layanan penting atau membayar langganan yang sudah tidak terpakai. 

**Fitur Utama yang Memudahkan Pengguna:**
* **Pencatatan Otomatis:** Memudahkan penambahan, pengeditan, dan penghapusan data langganan secara instan.
* **Indikator Status Cerdas:** Sistem otomatis mendeteksi dan mengubah status menjadi "Segera Berakhir" jika sisa waktu kurang dari 30 hari, mencegah keterlambatan perpanjangan.
* **Pencarian Cepat:** Menemukan data aplikasi atau departemen tertentu dalam hitungan detik.
* **Unduh Laporan (Excel/CSV):** Data dapat diekspor menjadi file CSV dengan satu klik untuk kebutuhan pelaporan keuangan atau audit.
* **Aman & Ringan:** Menggunakan *Local Storage* pada browser sehingga data tersimpan tanpa memerlukan database eksternal yang rumit.

## 🛠️ Keterampilan Teknis (Tech Stack)
Proyek ini dibangun untuk mendemonstrasikan pemahaman yang kuat tentang pengembangan *frontend* yang bersih, interaktif, dan responsif:

* **Antarmuka (UI/UX):** HTML5 & Tailwind CSS (Desain modern dan 100% responsif di HP maupun Laptop).
* **Logika Interaktif:** JavaScript murni (Manipulasi data, perhitungan tanggal otomatis, dan fitur *export*).
* **Deployment:** Di-hosting menggunakan Vercel untuk akses yang cepat dan stabil.

## 🚀 Cara Mencoba Aplikasi
Anda dapat langsung mengklik tautan *Live Demo* di atas untuk mencoba aplikasi ini langsung dari browser Anda tanpa perlu menginstal apa pun. Segala perubahan data yang Anda lakukan (tambah/edit) hanya akan tersimpan di perangkat Anda sendiri.
